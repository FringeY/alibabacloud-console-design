# @alicloud/xconsole-rc-gray

> 灰度组件，能根据 Viper 的灰度状态控制内容的展示和隐藏。

## APIs

### Gray

| Param | Type | Default | Description |
|--------|------|---------|-------------|
|id|`string`||灰度 id|

## Usage

```js
import Gray from '@alicloud/xconsole-rc-gray';

<Gray id="feature.cors.enable">
  <Component />
</Gray>
```
